Vector Addition — A Visual Intro for Coders

When working in games, graphics, or simulations, vectors are everywhere. But what are they, really?

What is a Vector?

A vector is a direction and magnitude — usually visualized as an arrow. It tells you how far to move, and in what direction.

In 2D code, a vector is typically written as:

let v = { x: 50, y: -20 };

This means: move 50 units right, 20 units up.

Adding Vectors

You can add two vectors by placing them tip-to-tail. The resulting vector connects the start of the first to the tip of the second.

let a = { x: 50, y: 0 };
let b = { x: 0, y: 50 };

let sum = { x: a.x + b.x, y: a.y + b.y };  // { x: 50, y: 50 }

Visually, it forms a right triangle — and the sum vector is the diagonal.

Why it Matters

Adding vectors lets you:

  • Combine multiple movements
  • Add forces (like gravity + jump)
  • Animate complex behaviors
  • Simulate physics or camera motion
